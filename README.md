# Citedy Claude Code Plugins

Official plugin marketplace by [Citedy](https://www.citedy.com).

## Add this marketplace

```bash
claude plugin marketplace add Citedy/claude-plugins
```

## Available plugins

| Plugin | Description |
|--------|-------------|
| **game-sounds** | Warcraft Peon voice lines for Claude Code |

## Install a plugin

```bash
claude plugin install game-sounds@citedy
```
